# johngoben


## Project setup

This project requires node v16.13.2.

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Lints and fixes files
```
npm run format
```

### Deploy to AWS
```
npm run deploy
```

### Troubleshooting
